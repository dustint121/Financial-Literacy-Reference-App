Financial Literacy Reference Site: About

The purpose of this website is to record the financial literacy information I have learned in a financial literacy course. I most certainly will not be able to retain all this information overtime, nor have access to the textbook long after the course.

Hence, I decided to store all the information by making a website using the basics of HTML, CSS and JavaScript to create a useful place for financial referencing and to practice the basic web development tools.

All the images on this site are provided by the book of reference for my financial literacy course, Personal Finance by Jack Kapoor.

For anyone happening to view my site, I hope that it helps you on your financial knowledge needs. - Dustin Tran
